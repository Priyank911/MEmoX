DevPilot Authentication Helper

This guide will help you complete the authentication process for DevPilot if the automatic redirect to VS Code didn't work.

Note: You should only need this page if VS Code didn't automatically open after GitHub authorization.

Method 1: Manual Code Entry

  1. Copy the authorization code from your browser's address bar. It appears after code= and before &state= or &
  2. Return to VS Code
  3. Click on the "DevPilot: Sign In" button in the status bar
  4. Select "Enter Auth Code"
  5. Paste the authorization code and press Enter

Method 2: Using a Personal Access Token

  1. Create a GitHub Personal Access Token at https://github.com/settings/tokens
  2. Ensure the token has the user:email scope
  3. In VS Code, click on the "DevPilot: Sign In" button in the status bar
  4. Select "Enter GitHub Token"
  5. Paste your Personal Access Token and press Enter
Tip: If you're experiencing persistent issues with the OAuth flow, you can try:
  • Restarting VS Code
  • Checking that the protocol handler is registered on your system
  • Verifying that your GitHub OAuth app has the correct callback URL: vscode://memox/callback

For additional help, please refer to the DevPilot Authentication Documentation.
